现在线上的环境只需要有openvino库就行吧，需要onnx库吗
Sunday 11:12 PM

线上跑我们现在这版 不需要 onnx 库。

原因是：

Stage3 CNN -> OpenVINO 走的是 openvino.convert_model(torch_model, example_input=...)
RawWave -> OpenVINO 也是直接从 PyTorch module 转 OpenVINO
我们没有在线上导出/读取 ONNX
Perch 仍然用 onnxruntime 跑原来的 perch_v2.onnx
所以线上需要：

必须有：openvino
必须有：onnxruntime
不需要：onnx
你可以在 Kaggle 里检查：

python -c "import openvino; import onnxruntime; print('ok')"
如果这句能过，就够了。
